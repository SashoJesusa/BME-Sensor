# ESP8266 Модул за следене на параметри на въздуха

Проектът представлява компактен електронен модул за наблюдение на основните параметри на въздуха в затворени помещения.

## Функционалност
* **Измерване:** Температура, влажност, налягане и качество на въздуха (газово съпротивление).
* **Визуализация:** Локална уеб страница, достъпна в реално време.
* **Smart Home:** Интеграция с Home Assistant чрез MQTT протокол.
* **Корекция:** Софтуерна температурна корекция (-10°C) за премахване на влиянието от загряването на ESP8266.

## Хардуер
* Микроконтролер: **ESP8266**.
* Сензор: **BME680** (високотехнологичен цифров сензор).
* Интерфейс: **I2C** (SDA -> D2, SCL -> D1).

## Структура на проекта
* src - Програмен код за Arduino IDE.
* docs - Пълно техническо описание и схематика.
* lib - Списък с необходимите библиотеки.
